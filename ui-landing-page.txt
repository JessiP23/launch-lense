SECTION 2 — DESIGN SYSTEM
2.1 — Philosophy

One rule above all: Every element earns its space by helping a user
make a decision or take an action. Decoration is a tax on clarity.


No gradients anywhere in the app (not even subtle ones)
No box-shadows (depth through border contrast and background layering only)
No icons in the app — all actions and states expressed in text/typography
No generic SaaS purple, blue, or teal — the only colors are ink, canvas, and signal colors
No rounded corners above 16px (landing page) / 12px (app)
Signal colors (go/warn/stop) used ONLY for data states — never decoratively

2.5 — Motion System
All animations use Framer Motion. Zero CSS animation except:

The nav scroll opacity transition (transition-all duration-300)
Button hover color transitions (transition-colors duration-150)
The live status ping dot (animate-ping — Tailwind only)

Standard easing: [0.16, 1, 0.3, 1] — fast out, physical. Use for EVERYTHING.
tsx// Page transition (wrap in AnimatePresence in layout.tsx)
const pageVariants = {
  initial: { opacity: 0, y: 6 },
  animate: { opacity: 1, y: 0, transition: { duration: 0.22, ease: [0.16, 1, 0.3, 1] } },
  exit:    { opacity: 0, transition: { duration: 0.15 } }
}

// Scroll reveal (useInView + variants)
const fadeUp = {
  hidden: { opacity: 0, y: 24 },
  show:   { opacity: 1, y: 0, transition: { duration: 0.55, ease: [0.16, 1, 0.3, 1] } }
}

// Stagger wrapper
const stagger = (delay = 0.07) => ({
  show: { transition: { staggerChildren: delay } }
})

// Score ring (SVG strokeDashoffset)
// Animate from circ → circ - (score/100 * circ)
// duration: 1.4s, ease: "easeOut", delay: 0.2

// Wizard step transitions
const stepIn  = { opacity: 0, x:  20 }
const stepOut = { opacity: 0, x: -20 }
// duration 0.22s each direction

// Card hover (ONLY on clickable cards)
whileHover={{ scale: 1.015, transition: { duration: 0.18 } }}
// NEVER scale on buttons — color change only

try { ... } catch (e) {

  return Response.json({ error: 'Internal error' }, { status: 500 })
}

SECTION 9 — COPY STYLE GUIDE
Headlines: Short, declarative, anti-hype. Never exclamation marks.
Sub-copy: One idea per sentence. Specific numbers always outperform vague claims.
Labels: ALL CAPS, tracked, short (≤ 2 words)
CTAs: Action verb first. Specific > generic. "Run one test" > "Get started"
Error messages: State what happened + what to do. "Token expired. Reconnect your account."
Verdict copy: Clinical. "CTR exceeded 2% threshold. GO signal confirmed." Not emotional.
